SoulsArmoryAPI

Es una API REST que en la que puedes obtener información de armas de este hermoso jueguito que tanta bronca nos puede dar a veces 🤬.

También puedes agregar, editar y borrar (no las borres porfa) nuevas armas o escudos 😘.

¡Hola!
Hice esta API, creada con Spring Boot, que proporciona información detallada sobre las armas del juego Dark Souls 1. No tiene fines de lucro ni comerciales, simplemente es un proyecto personal para practicar y mejorar mis conocimientos en el desarrollo de API REST.
Toda la información contenida en esta API ha sido obtenida de la Wiki de Dark Souls (tremenda página). No tengo intención de apropiarme de su contenido, sino que lo utilizo con fines educativos y como parte de un ejercicio de desarrollo.

¡Información!
Normalmente, todos los nombres empiezan con Mayúscula la primera letra. Puedes escribirme para cualquier consejo o sugerencia.
Lo aclaro más abajo, pero bueno, no está de más repetirlo. Si quieres usar algún endpoint de los que obtienen armas a partir del nombre, cuando este contenga espacios, como por ejemplo "Espada recta de Astora", puedes buscarla poniendo "Espada_recta_de_Astora" o "Espada%20recta%20de%20Astora".
Arriba hay un apartado para subir más armas si quieres, o también puedes hacerlo mediante solicitudes HTTP, aunque es más engorroso.
